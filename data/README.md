# Voice to Pin - YalaReady

ฟีเจอร์แปลงข้อความจากเสียงภาษาไทยให้เป็นพิกัดสำหรับปักหมุดบนแผนที่ โดยเน้นพื้นที่เมืองยะลา

ระบบนี้ใช้หลายแหล่งข้อมูลร่วมกัน:

1. Local gazetteer จาก OSM
2. ฐานข้อมูลซอย/ถนนย่อย `geo_sois.csv`
3. Longdo Extract Address สำหรับแยกบ้านเลขที่ ถนน ซอย จังหวัด
4. Manual confirmation สำหรับเคสที่ยังไม่มั่นใจ

# Data files

## geo_sois.csv

ฐานข้อมูลซอย/ถนนย่อยในพื้นที่ยะลา มีคอลัมน์หลัก:

- `name`
- `parent_road`
- `name_en`
- `highway`
- `lat`
- `lng`

ใช้โดย `scripts/import_geo_sois.py` เพื่อ merge เข้า `data/yala_gazetteer.sqlite`

## yala_gazetteer.sqlite

SQLite database สำหรับ local matching

ตารางหลัก:

`gazetteer`

คอลัมน์สำคัญ:

- `name`
- `name_clean`
- `source_type`
- `lat`
- `lon`
- `name_key`
- `osm_type`
- `osm_id`

source type ที่ใช้ตอนนี้:

- `school`
- `hospital`
- `place_of_worship`
- `poi`
- `road`
- `soi`
